# TaskFlow - Enterprise Project Management SaaS

## Features

- JWT Authentication
- Workspace Management
- Kanban Board
- Real-Time Notifications
- Task Assignment
- Analytics Dashboard
- PDF/Excel Export
- AI Task Assistant
- Subscription System
- MERN Stack Architecture

## Tech Stack

Frontend:
- React
- Vite
- Context API
- Socket.io Client

Backend:
- Node.js
- Express.js
- MongoDB
- JWT
- Socket.io

Deployment:
- Vercel
- Render
- MongoDB Atlas

## Installation

### Frontend

npm install
npm run dev

### Backend

npm install
npm start
